Tools & Technology

Everything powering SentinelPass — from the Python backend and cryptographic libraries to the frontend design system and DevOps workflow.

Technology Stack

SentinelPass is built with a modern, security-conscious full-stack approach.

Backend

🐍
Python 3.11+
Core language. Handles cryptography, entropy math, and API logic.
⚗️
Flask 3.x
Lightweight WSGI micro-framework for routing and Jinja2 templating.
🔐
secrets module
Python's CSPRNG for cryptographically secure password generation.
🔑
hashlib (SHA-1)
Hashes passwords locally for k-Anonymity HIBP breach lookups.
📐
math (log₂)
Shannon entropy calculation: H = L × log₂(R).
🌐
urllib / HIBP API
Privacy-preserving breach checking against 750M+ leaked passwords.

Frontend

🌐
HTML5
Semantic markup, ARIA roles, and Jinja2 template inheritance.
🎨
Vanilla CSS3
Custom properties, glassmorphism, CSS Grid, animations, and responsive design.
⚡
Vanilla JavaScript
Fetch API, DOM manipulation, Canvas particle system, debouncing.
✨
Lucide Icons
Clean SVG icon library loaded via CDN — lightweight and crisp.
🔤
Google Fonts
Outfit (UI) + Fira Code (monospace) for a premium typographic feel.
🎭
Canvas API
Animated particle background rendering using HTML5 Canvas 2D context.

DevOps & Tools

🐙
GitHub
Version control, collaboration, and open-source publishing.
📦
pip / venv
Python dependency management and isolated virtual environments.
🔒
HIBP k-Anonymity
Zero-trust privacy model — passwords never leave your device in full.
🧪
REST API
JSON endpoints: /api/analyze and /api/generate with error handling.

Key Features

  • Real-time Shannon entropy calculation (H = L × log₂R)
  • k-Anonymity breach checking via HIBP API (750M+ hashes)
  • Crack time estimation across 3 attack models
  • Character distribution visualisation
  • Server-side CSPRNG password generation (Python secrets)
  • Memorable passphrase generator with custom separators
  • Live brute-force attack simulator with real-time console
  • Interactive dictionary attack checker
  • 7-topic expandable cybersecurity knowledge base
  • Session-based password generation history
  • Canvas particle animation background
  • Fully responsive mobile-first design
